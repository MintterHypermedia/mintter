# Button

The `Button` component can be used to initiate an action, like can be submitting
a form or anything you want.

## Import

```tsx
import {Button} from '@mintter/ui/button'
```

## Usage

```jsx
<Button>This is my Button!</Button>
```

### Size

Pass a `size` prop to the `Button` component to change their size.

```jsx columns
<Button size="1">Small Button</Button>
<Button size="2">Medium Button</Button>
<Button size="3">Large Button</Button>
```

> The default `Button` size is `medium`.

### Variant

Pass a `variant` prop to the `Button` component to change their appearance.

```jsx columns
<Button variant="solid">Solid Button</Button>
<Button variant="outlined">Outlined Button</Button>
<Button variant="ghost">Ghost Button</Button>
```

> The default `Button` variant is `solid`.

### Shape

Pass a `shape` prop to the `Button` component to change its shape.

```jsx columns
<Button shape="rounded">Rounded Button</Button>
<Button shape="pill">Pill Button</Button>
```

> The default `Button` `shape` is `rounded`.

### Color

The `Button` component has various color schemes based in the Theme brand and
status colors.

```jsx columns
<Button color="primary">Primary button</Button>
<Button color="secondary">Secondary button</Button>
<Button color="terciary">Terciary button</Button>
<Button color="success">Success button</Button>
<Button color="warning">Warning button</Button>
<Button color="danger">Danger button</Button>
```

> The default `Button` color is `primary`.
