# Alert

The `Alert` object exposes a set of
[accessible components](https://radix-ui.com/primitives/docs/components/alert-dialog) that interrupts user interactions
with important content.

## Import

```tsx
import {Alert} from '@mintter/ui/dialog'
```

## Usage

```jsx
<Alert.Root>
  <Alert.Trigger>Open</Alert.Trigger>
  <Alert.Content>
    <Alert.Title>Alert Title</Alert.Title>
    <Alert.Description>Alert Description</Alert.Description>
    <Alert.Actions>
      <Alert.Cancel>Cancel</Alert.Cancel>
      <Alert.Action>Accept</Alert.Action>
    </Alert.Actions>
  </Alert.Content>
</Alert.Root>
```
