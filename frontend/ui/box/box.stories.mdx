# Box

The `Box` component is one of our most basic primitives, use it and combine it
with other components to build more complex UI patterns.

## Import

```tsx
import {Box} from '@mintter/ui/box'
```

## Usage

```jsx
<Box>Look ma, a Box!</Box>
```
